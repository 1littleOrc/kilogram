What is it?

Kilogram is a Gulp based workflow for easy and clean email development

Kilogram is the best choice for most responsive email templates. Code based on Fluid Method.

Usage

  • Download Kilogram
  • Install dependencies via npm install
  • Go to project folder and run gulp
  • Have fun!

Why?

I have to create simple pages in my daily work. Sometimes it's compiled Markdown and sometimes it's pure HTML . I'd like to make them more beautiful without additional classes, ids, etc. So I did it.
